VM IP Address -
Node IP -
Username ubuntu

Connect through the node as a jump host. This is the quickest way to access your VM. 

ssh -J root@NODE_IP ubuntu@VM_IP

Steps:

1
Ensure your SSH public key was added to your DeCloud account before creating this VM.
2
Run the SSH command above from your terminal. The -J flag uses the node as a jump host.
3
If prompted about host authenticity, type yes to continue.

Set up WireGuard for direct access to your VM without using a jump host.

WireGuard Endpoint -
Hub Public Key -

Setup Steps:

1
Install WireGuard on your machine: brew install wireguard-tools (Mac) or download from wireguard.com
2
Generate your keypair: wg genkey | tee privatekey | wg pubkey > publickey
3
Create config file wg-decloud.conf with your private key and the hub's public key shown above.
4
Contact the node operator to add your public key as a peer.
5
Activate WireGuard: wg-quick up wg-decloud
6
Connect directly: ssh ubuntu@VM_IP
